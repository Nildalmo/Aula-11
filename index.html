Aula Git e Github

git init

Inicializar um novo repositório Git em um diretório.

git status

Exibe o status atual do seu repositório. mostra quais arquivos foram modificados, adicionados e excluidos.

git add. ou git add 'nome do arquivo'

Adiciona as alterações de um arquivo, preparando-o para o proximo commit

git commit

Grava uma alteração do diretório. A messagem do commit deve descrever as alterações realizadas.

git log

Mostra o historico de commits do repositório, exibindo informações como autor, data e menssagem
